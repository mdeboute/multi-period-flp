Organization of the instance file
 * First line: |I| |J| |T|
 * Second line: Values of p^t for t\in T
 * Third line: Values of n^t for t\in T
 * From fourth line: A block of lines for every i\in I
 	* First line of each block: values of f^t_i for t\in T
 	* Then, a line for every j\in J : values of c^t_ij for t\in T
